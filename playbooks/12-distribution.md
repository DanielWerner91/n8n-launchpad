# Phase 12: Distribution

> Pipeline stage: live
> Prerequisites: Phase 11 (Landing Page) complete
> Skills: /linkedin-strategy, /newsletter-builder, /carousel-generator
> Estimated items: 3 checklist items

## Overview

Launch the content distribution machine. This is where the app starts getting traffic. LinkedIn is the primary channel, supplemented by newsletter and directory listings.

## Checklist Items

### [distribution] LinkedIn launch content
**Approach:** Follow `/linkedin-strategy` (THE BIBLE) for all LinkedIn work. Create launch content:

1. **Launch post:** Carousel format (1.45x engagement multiplier). 5-7 slides covering: problem, solution, key features, social proof, CTA
2. **Follow-up posts:** Schedule 3-5 posts over the first week covering different angles
3. **Comment strategy:** Engage on relevant posts in the niche with value-added comments

LinkedIn rules (non-negotiable):
- Zero hashtags
- Links in first comment only, never in the post body
- Carousels are king (always prefer over text posts)
- Golden window: first 60 minutes determine reach
- 8-hour minimum gap between posts
- Optimal posting: Tue-Thu 9-11AM (target user's timezone)
- Use brand voice from Content Flywheel

Generate carousels using `/carousel-generator`. Generate infographics using `/infographic-generator` for data-driven content.
**Skill:** `/linkedin-strategy`, `/carousel-generator`
**Done when:** Launch carousel is created and posted. At least 3 follow-up posts are scheduled. First-week engagement is monitored.
**References:** []
**Depends on:** Landing page live, Brand voice defined

### [distribution] Beehiiv newsletter set up
**Approach:** Set up a newsletter using `/newsletter-builder`:
1. Create a Beehiiv publication at beehiiv.com (free tier)
2. Configure with the app's brand name and colors
3. Set up the TLDR newsletter format
4. Add a newsletter signup to the landing page and app
5. Create the first edition announcing the launch

The newsletter pipeline: Beehiiv for sending, Content Flywheel for content generation, `/newsletter-builder` for formatting.
**Skill:** `/newsletter-builder`
**Done when:** Beehiiv publication exists. Newsletter signup is on the landing page. First edition is drafted.
**References:** ["https://beehiiv.com"]
**Depends on:** Landing page live, Content Flywheel integrated

## Decision Points

- Personal brand or company brand for LinkedIn? (Daniel Werner personal for most apps)
- Product Hunt launch timing? (Aim for Tuesday-Thursday)
- Which directories are relevant for this specific niche?

## Common Pitfalls

- **Posting links in LinkedIn post body.** Links go in the FIRST COMMENT only. This is non-negotiable.
- **Using hashtags on LinkedIn.** Zero hashtags. The algorithm penalizes them.
- **Launching on Friday.** Tuesday-Thursday gets 2-3x more visibility.
- **One-and-done posting.** The launch needs sustained content over weeks, not a single post.
